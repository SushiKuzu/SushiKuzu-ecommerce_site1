```markdown
# ecommerce_site1

A demo e-commerce website scaffolded with Next.js, Prisma (SQLite), NextAuth, and Stripe (test mode).

Setup
1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env.local` and fill the values
3. Run Prisma generate + seed: `npx prisma generate && node prisma/seed.js`
4. Run dev server: `npm run dev`

Environment variables (see .env.example)
```
